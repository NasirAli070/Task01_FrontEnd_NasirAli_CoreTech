# Task 01: Front-End Development - CoreTech Innovations

## Project Overview
This is a client-ready sample webpage created for CoreTech Innovations as part of a front-end development internship task. The webpage showcases modern web development practices with responsive design and interactive elements.

## Features
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**:
  - Color theme customizer
  - Visitor counter with increment/decrement functionality
  - Mobile-friendly navigation menu
  - Hover effects and animations
- **Modern UI Components**:
  - Hero section with call-to-action
  - Services showcase
  - Contact information footer
  - Social media integration

## Technologies Used
- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript (no frameworks)
- Font Awesome icons
- Google Fonts (via system fonts)
